De VIVES website gebruikt een aparte css stylesheet die werd verwezen door middel van een link. Ze maken ook gebruik van inline css en het id element.

De MIRAS website gebruikt een aparte css stylesheet die werd verwezen door middel van een link. Ze maken ook gebruik van class en id, dus ze gebruiken style blokken in de body.

De VRT website gebruikt een style blok in de head van de pagina. Ze maken ook gebruik van het class element.
